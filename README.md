# NS-RobotArm Main Repository

## Software

**System requirements:**

Ubuntu 18.04 LTS

Recommended:

Realtime kernel: https://mirrors.edge.kernel.org/pub/linux/kernel/projects/rt/

**Install:**

```
sudo apt-get install git

git clone https://github.com/Noothless/NS-RobotArm.git

cd ~/NS-RobotArm

./install.sh
```
